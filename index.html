Estrutura Basica de um Documento

Um documento HTML é formado por tags que definem os elementos do conteúdo. Os elementos podem ser divididos em blocos e indivíduos.

Um documento HTML típico possui a seguinte estrutura básica:

Vamos decompor cada parte:


        <!DOCTYPE html>: Esta declaração define o tipo de documento e a versão do HTML que está sendo usada. No caso, é HTML5.
        
        <html>: Esta tag é o contêiner raiz para todo o conteúdo HTML.
        
        <head>: A seção <head> contém meta-informações sobre o documento, como o título da página (<title>), links para estilos CSS, scripts JavaScript, entre outros.
        
        <title>: Define o título da página que aparece na aba do navegador.
        
        <body>: A seção <body> contém todo o conteúdo visível da página, como texto, imagens, links, vídeos, etc.

Principais Tags HTML


        <h1> a <h6>: Tags de cabeçalho, com <h1> sendo o mais importante e <h6> o menos importante.
        
        <p>: Define um parágrafo.
        
        <a>: Define um hiperlink.
        
        <img>: Insere uma imagem.
        
        <ul>, <ol>, <li>: Cria listas não ordenadas (<ul>) e ordenadas (<ol>) com itens de lista (<li>).
        
        <div>: Define uma seção ou divisão no documento. É uma tag de contêiner.
        
        <span>: Semelhante ao <div>, mas é usado para marcar uma parte do texto ou conteúdo inline.
        
        <table>, <tr>, <td>: Utilizadas para criar tabelas.

Atributos HTML

Tags HTML podem ter atributos que fornecem informações adicionais sobre o elemento. Alguns atributos comuns incluem:
id: Um identificador único para o elemento.
class: Define uma classe ou grupo de elementos que podem ser estilizados juntos.
src: Especifica a fonte de um recurso externo, como uma imagem (<img>) ou um script (<script>).
href: Especifica o URL de um link ().
alt: Texto alternativo para uma imagem, utilizado quando a imagem não pode ser exibida.

Considerações Finais

HTML é uma linguagem fundamental para a web e serve como a espinha dorsal de qualquer site. Aprender HTML é o primeiro passo para se tornar um desenvolvedor web, e uma boa compreensão de como ele funciona é essencial para criar páginas web eficazes e funcionais
